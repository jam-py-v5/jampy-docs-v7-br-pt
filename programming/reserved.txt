===================
Palavras reservadas
===================

Como o Jam.py é um framework que utiliza as linguagens Python e JavaScript, 
devem ser usados identificadores válidos de Python e JavaScript.  
Além disso, para todos os bancos de dados suportados, devem ser utilizados identificadores válidos de acordo com cada banco.

O arquivo que contém uma lista de exemplos é ``keywords.py``, com o conteúdo abaixo:

.. code-block:: py

	keywords = [
		"BADGE",
		"LABEL",
		"HIDDEN",
		"PROGRESS",
		"TASK"
	]
